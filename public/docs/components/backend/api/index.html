C3S API

C3S API v3.1.0

Scroll down for code samples, example requests and responses. Select a language for code samples from the tabs above or the mobile navigation menu.

An API for creating, running and analysing citizen science projects

Base URLs:

  • https://staging.citizenscience.ch/v3

  • https://api.citizenscience.ch/v3

Authentication

  • API Key (anonUser)

    • Parameter Name: X-ANON, in: header.

  • API Key (apiKeyHeader)

    • Parameter Name: X-API-KEY, in: header.

  • oAuth2 authentication.

    • Flow: implicit
    • Authorization URL = http://localhost:8081/oauth/authorize
Scope Scope Description
apiKey Unique identifier of the user accessing the service.
